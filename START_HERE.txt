                    📋 QUICK START - 4 SIMPLE STEPS
================================================================================

STEP 1: CLOUDFLARE DNS
-----------------------
Login to Cloudflare → bsapservices.com → DNS
Add A record: claim → [YOUR VPS IP] → Proxied ON
SSL/TLS: Set to "Full"


STEP 2: UPLOAD FILES
--------------------
Windows PowerShell:
scp -r C:\Users\nfectious\Downloads\claim root@YOUR_VPS_IP:/root/

(Replace YOUR_VPS_IP with actual IP)


STEP 3: SSH & DEPLOY
---------------------
ssh root@YOUR_VPS_IP

Then run:
mv /root/claim /var/www/herd-case
cd /var/www/herd-case
chmod +x deploy.sh
sudo bash deploy.sh

Press Enter when prompted. Wait 10-15 minutes.


STEP 4: DONE!
-------------
Visit: https://claim.bsapservices.com

Your website is LIVE! 🎉
